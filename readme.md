
# Patrikos — Terminal Based IDE (Beta)

## Introduction

Patrikos is a terminal-based IDE / lightweight text editor built from scratch using **C** and **ncurses**.

This project is mainly a learning journey to understand:
- how text editors work internally
- low-level input handling
- terminal UI rendering
- file systems in C

I created it because tools like  nvim and micro feel powerful but overwhelming at first. Patrikos is my attempt to build something simpler from the ground up while learning C in the process.

---

##  Features (Beta)

Right now, Patrikos is still in development and only supports basic functionality:

1.  Text input (typing)
2.  Text deletion
3.  Cursor movement (arrow keys)
4.  File opening
5.  File saving

---

## ⌨ Shortcuts

- `Ctrl + S` → Save file  
- `Ctrl + O` → Open file  
- `Ctrl + Q` → Quit editor  

---

##  Tech Stack

This project is built using:

- C (vanilla)
- ncurses library for terminal UI
